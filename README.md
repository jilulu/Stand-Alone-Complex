## Actions 
* **Show available tasks** UNIX `./gradlew tasks`, or Windows `gradlew.bat tasks`. 
* **Run the web app** UNIX `./gradlew appRun`, or Windows `gradlew.bat appRun`.
* **Run tests** UNIX `./gradlew test`, or Windows `gradlew.bat test`.
